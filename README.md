# Logic Gallery

A premium logic gallery website with interactive features.

## Features

- **Home Page**: Welcome page with animated particle background
- **Portfolio Page**: 5 project cards showcasing different applications
- **Favorites Page**: 8 artwork cards with like and comment functionality
- Interactive particle.js background
- Glassmorphism design
- Like/Unlike artwork with heart icon
- Comment system for each artwork
- Responsive navigation

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, Animations)
- JavaScript (Vanilla JS)
- Particles.js

## Pages

1. **index.html** - Home/Landing page
2. **portfolio.html** - Portfolio showcase with 5 projects
3. **favorites.html** - Gallery with 8 artworks

## Setup

1. Clone the repository:
```bash
git clone https://github.com/skbeltranpogi/tomol.git
```

2. Open `index.html` in your browser or use Live Server

## Usage

- Navigate between pages using the top navigation bar
- Click heart icons to like/unlike artworks
- Add comments to any artwork
- Interact with the particle background (hover and click)
